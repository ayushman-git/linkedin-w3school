Semantic tags and their uses.

  • Main - This tag is used once in a whole page. It declares the main content of the page. Like main banner, main article etc.
  • Section - Section is used to differentiate between components in a page. For eg. in a one page website, there's main tag which is a banner then there's information about the service the website provide, it will be wrapped in a section tag then there's contact info which should be wrapped in another section tag.
  • Article - Article tag is used to wrap a content like a blog post, a card etc. They tells us that this is a content which can be useful for the user.
  • Footer - Footer tag is used to wrap the additional information of an article or actual footer of the page. We can use it in a blog post for social media links.
  • Aside - This tag is used for information that show up on the sides of the page. Like a side navabar.
