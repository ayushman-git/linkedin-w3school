Generic Tags

The span is an inline generic tag which means we can use it anywhere in between lines and characters.

Div is a block element which takes the whole width of the page. It is used to group a tags but it don't have a semantive value.
That's why we should always try to use semantic tags and use <span> and <div> as a last resort.
